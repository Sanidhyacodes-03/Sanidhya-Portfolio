# рџљЂ Installation

## Clone Repository

```bash
git clone https://github.com/Sanidhyacodes-03/Sanidhya-Portfolio.git

cd Sanidhya-Portfolio
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

Runs at

```
http://localhost:5173
```

---

## Backend Setup

```bash
cd server

npm install

npm start
```

Runs at

```
http://localhost:5000
```

---

# рџ”‘ Environment Variables

Create a `.env` file inside the `server` folder.

